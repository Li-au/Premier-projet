# Product Guide — Weather & Time Dashboard

## Vision
Une page web personnelle publiée sur GitHub Pages affichant en temps réel 
la météo, l'heure française et l'heure de New York. Premier projet de stage, 
conçu pour être partagé publiquement.

## Utilisateurs cibles
- L'auteur du projet (usage personnel / vitrine de stage)
- Toute personne visitant le lien GitHub Pages

## Fonctionnalités clés
1. **Météo en temps réel** — L'utilisateur saisit une ville et obtient :
   - Température actuelle
   - Description des conditions (ensoleillé, nuageux, pluie…)
   - Icône météo animée
2. **Horloge France** — Heure en direct (Paris, Europe/Paris)
3. **Horloge New York** — Heure en direct (America/New_York)
4. **Mise à jour automatique** — Les horloges s'actualisent chaque seconde

## Objectifs
- Projet vitrine pour un stage (premier projet web)
- Déployé sur GitHub Pages (statique, sans backend)
- Code propre et lisible pour démontrer les compétences acquises

## Contraintes
- Aucun backend : utilisation d'une API météo publique (ex: OpenWeatherMap)
- Technologie : HTML / CSS / JavaScript vanilla (pas de framework)
- Hébergement : GitHub Pages (fichiers statiques uniquement)
