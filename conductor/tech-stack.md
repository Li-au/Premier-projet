# Tech Stack — Weather & Time Dashboard

## Langages
- HTML5 (structure de la page)
- CSS3 (styles et animations)
- JavaScript ES6+ (logique, API, horloges)

## API externe
- **OpenWeatherMap** — API météo gratuite (plan Free, 1000 appels/jour)
  - Endpoint : `https://api.openweathermap.org/data/2.5/weather`
  - Clé API requise (gratuite sur openweathermap.org)

## Hébergement
- **GitHub Pages** — Déploiement statique gratuit depuis la branche `main`

## Outils
- Git / GitHub (versioning et déploiement)
- Navigateur web (test local via `file://` ou Live Server)

## Dépendances
- Aucune bibliothèque externe obligatoire
- Vanilla JavaScript uniquement (pas de npm, pas de framework)
