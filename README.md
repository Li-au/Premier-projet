# 🌤️ Une Info — Dashboard Météo

Dashboard météo en temps réel avec effets visuels animés selon la température.

**👉 [Voir en ligne](https://li-au.github.io/Premier-projet/)**

## Fonctionnalités

- **Météo en temps réel** — température, description et icône via Open-Meteo (sans clé API)
- **Photo de la ville** — image récupérée automatiquement depuis Wikipedia
- **Effets animés** selon la température :
  - Neige et flocons pour les temps froids
  - Vent pour les températures fraîches
  - Bulles pour les températures douces
  - Flammes pour la chaleur (plus intenses au-delà de 35°C)
- **Horloges en direct** — Paris et New York avec date complète
- **Suggestions intelligentes** — propose des villes alternatives si tu fais une faute de frappe, sans doublons par pays

## Comment utiliser

Tape le nom d'une ville et appuie sur **Entrée**. Le fond de page change de couleur selon la température.

## Technologies

HTML · CSS · JavaScript (vanilla) · [Open-Meteo API](https://open-meteo.com/) · Wikipedia API
